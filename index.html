Raspcam

This is a simple script which uses fswebcam and apache to make a simple raspberry pi security camera.

Download this project as a .zip file Download this project as a tar.gz file

This is a simple script which uses fswebcam and apache to make a simple raspberry pi security camera. It will save pictures to IPADRESS/camera.jpg, and will save pictures with the DATE as name (IP/YEAR-MONTH-DAY-HOURMINUTE). For example, my raspberry pi picture from 2014-8-8 12:21 Will be saved as: http://192.168.0.110/2014-08-08_1221.jpg

What you need

  • Raspberry Pi with Raspbian and connection to the internet
  • Camera (I use an Playstation Eye)

Installing a web server

First we need to install a web server on your raspberry pi, you can do this by typing this command:

sudo apt-get install apache2 Restart it

sudo service apache2 restart Then install fswebcam, with fswebcam you can make pictures with your default USB webcam.

sudo apt-get install fswebcam

Downloading the script

Install git. Type this in your command line:

sudo apt-get install git Now download the script

git clone DOWNLOAD_LINK_COMING raspcam.sh We need permission to execute the file, to do this we need to add 'X' to our permissions, type in:

sudo chmod +x raspcam.sh Now we are going to run the file, make sure you have your USB camera connected on the USB ports on the raspberry pi, not on a powered USB hub:

./raspcam.sh Press Ctrl+C to stop the script

Viewing your pictures

To view our pictures, we first need to know the IP Address of our Raspberry Pi. Type in:

ifconfig You will see something simular to this: We need to know the numbers after 'inet adress:', the will look simular to '192.168.0.110' (IP adress of my raspberry pi) Now enter the IP Adress of your Raspberry Pi in your browser, and add '/camera.jpg'. Now press enter. You will see the last picture taken by your camera! To see the latest picture (RaspCam will take a picture every minute) press F5.
